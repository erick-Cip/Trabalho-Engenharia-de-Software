Projeto: Exemplo Facade - Sistema de Cursos Online
Autor: Érick Cipriano Gonçalves

Como compilar e executar (Java 11+):

1) Abra um terminal na pasta onde estão os arquivos .java
2) Compile: javac *.java
3) Execute: java Main

Arquivos incluídos:
- UsuarioService.java
- CursoService.java
- PagamentoService.java
- CertificadoService.java
- CursoFacade.java
- Main.java
